# AI Interview Portal

## Overview
AI Interview Portal is a full-stack web application built using Flask, MySQL, HTML, CSS, and JavaScript. It enables users to practice technical interview questions, track performance, and manage assessments through an admin panel.

## Features
- User Registration & Login
- Admin Panel for Question Management
- Dynamic MCQ-Based Assessments
- Automatic Score Calculation
- Result History Tracking
- MySQL Database Integration
- REST API Backend using Flask

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- Database: MySQL
- Version Control: Git & GitHub
